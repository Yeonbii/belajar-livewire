## Pengingat Bagi Pelupa
- Setelah clone, jalankan command ini
```sh
cd belajar-livewire
composer install
cp .env.example .env
php artisan key:generate
```
- Sesuaikan file .env
- Setelah itu lanjutkan ini
```sh
php artisan migrate
npm install
npm run build
```
- Jalankan projectnya
```sh
composer run dev
```
